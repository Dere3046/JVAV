Conceived by Dr. Zhang Haoyang in 1025 AD, now maintained by the JVAV open source community. Features a complete compiler, assembler, virtual machine, standard library, and the innovative MimiWorld ownership system.

快速开始Quick Start

1. 编写源码

2. 一键编译运行

jvlc --run hello.jvl

3. 查看输出

42
